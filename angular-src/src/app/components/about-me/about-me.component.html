Employee Management App



My name is David Maman, I am a developer and this is my second project.
My first project was an application to send messages to Android. A very challenging project where I learned a lot about how to work on a server and client side. The project was developed on a server side in PHP and a client side in JAVA.

About 'Employee Managament':

For a few years I have been working as a security guard, currently working at the 'President's Residence'.
Working as a security man means working shifts during the week. Until a few years ago employee management was manually, employees send to the manager what they can / can not work during a week and the manager tries to embed by that. If this is a small amount of workers then there is no problem, but if it is a lot of employees placement can be very problematic. Here my project, try to provide management solution
For this project I work with NodeJS, Express, Mongodb, MLab, Angular and Heroku.

How It Works:

Registration is solely for the employee.
To register you need to contact the manager.
After log in there are several options and administrator notifications.
Option in Home Page:

  • Profile page
  • The Edit Personal Details page on this page does not require updating your password.
  • Contact page: List of contacts in the system records. Mailing option is currently inactive.
  • Week page: Here the employee is shown the number of shifts required for "opening".
    • To receive admin privileges please contact me by e-mail.
